IPL Data Analysis Project 🏏
This project performs Exploratory Data Analysis (EDA) on the IPL dataset using Python.
What's inside?
Visualizations: Bar charts and pie charts of match results and player performances.
Top Players: Analysis of "Player of the Match" winners.
Winning Trends: Distribution of wins by runs and wickets.
Tools Used:
Pandas, Matplotlib, Seaborn, and Google Colab.
